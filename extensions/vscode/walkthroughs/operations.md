Use the operational commands to manage local PairOfCleats workflows from VS Code:

- `PairOfCleats: Index Build`
- `PairOfCleats: Start Index Watch`
- `PairOfCleats: Index Validate`
- `PairOfCleats: Start Service API`
- `PairOfCleats: Start Service Indexer`

Long-running command output is streamed into the `PairOfCleats` output channel, and active runs stay visible through workflow status commands.
