Use the operational commands to manage local PairOfCleats workflows from VS Code:

- `PairOfCleats: Setup`
- `PairOfCleats: Bootstrap`
- `PairOfCleats: Tooling Doctor`
- `PairOfCleats: Config Dump`
- `PairOfCleats: Index Health`
- `PairOfCleats: Index Build`
- `PairOfCleats: Start Index Watch`
- `PairOfCleats: Index Validate`
- `PairOfCleats: Start Service API`
- `PairOfCleats: Start Service Indexer`

Short-lived diagnostic commands render structured summaries with explicit next actions in the `PairOfCleats` output channel. Long-running command output is streamed there as well, and active runs stay visible through workflow status commands.
